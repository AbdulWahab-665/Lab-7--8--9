About

This page uses Flexbox to structure the main layout and media queries at 600px and 900px to adjust flow for mobile, tablet, and desktop.

  • Mobile (≤600px): stack header, content, sidebar, footer vertically.
  • Tablet (601–899px): content and sidebar stacked, wider content area.
  • Desktop (≥900px): two-column layout with sticky sidebar.
